Towards realistic long-form podcasts with dialectal and paralinguistic diversity

This page presents an anonymized demonstration of a speech generation system for multi-turn, multi-speaker podcast dialogue. It highlights cross-dialect generation, zero-shot speaker conditioning, and controllable paralinguistic events.

Overview

What reviewers can evaluate

01

Multi-speaker podcast synthesis

Scripts are written as alternating speaker turns, allowing the generated audio to preserve podcast-like conversational flow.

02

Dialectal generation

The same prompt voices can be conditioned to produce Mandarin, Sichuanese, Cantonese, and Henanese dialogue examples.

03

Paralinguistic controls

Inline control tags such as laughter and sighs are represented in the input text for more expressive synthesized speech.

Review Notes

Anonymization checklist

  • No author names, affiliations, emails, QR codes, or personal links are shown.
  • No public repository, organization, model-hosting, or paper links are embedded.
  • Local assets are referenced with relative paths so the directory can be zipped or statically hosted.
  • Generated audio can be added without changing page code by keeping the expected filenames.
